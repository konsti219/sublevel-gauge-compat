# Sublevel Gauge Compat

Compatibility fixes for Create factory gauges and Deployer panels on Sable sub-levels, with a few targeted improvements for related addons.

Requires:
- Create
- Sable
- Deployer

## Features

- Makes Create factory panels and links behave correctly on Sable sub-levels, including connection targeting and sync-sensitive interactions.
- Improves Redstone Link compatibility for Deployer panels, including support for number-based signals.
- Extends Extra Gauges expression panels with comparison logic, conditional expressions, longer formulas, and redstone output from non-zero results.
- Improves Extra Gauges integer memory mode by preserving decimal values instead of truncating them.
- Keeps linked panels updating reliably when dynamic Simulated Gauges or Simulated sensor values change.
